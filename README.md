# 🛒 E-Commerce Order Management System (EcoMS)

## 📌 Overview

The E-Commerce Order Management System (EcoMS) is a web-based platform that allows customers to browse products, place orders, and track order status, while providing administrators with full control over products, inventory, and orders.

---

## 🎯 Objectives

* Simplify online shopping experience for customers
* Provide real-time inventory and order management
* Deliver a scalable and efficient backend system
* Enable admins to manage products, orders, and reports easily

## 🚀 Features

### 👤 Customer Features

* User registration and login
* Browse and search products
* View product details
* Add to cart and manage items
* Checkout and place orders
* Order history and status tracking

### 🛠️ Admin Features

* Admin dashboard
* Product management (Add/Edit/Delete)
* Inventory management
* Order management
* Automated reports

---

## 🏗️ Tech Stack

| Layer          | Technology           |
| -------------- | -------------------- |
| Frontend       | React + Tailwind CSS |
| Backend        | Node.js + Express.js |
| Database       | MongoDB              |
| Authentication | JWT + Bcrypt         |
| Testing        | Vitest               |
| Caching        | Redis                |

---

## 🔄 Workflow

1. Customer browses products
2. Adds items to cart
3. Proceeds to checkout
4. Places order
5. Admin manages order and inventory
